# Antivirus
Small school project game

VERSION TNSI2024 20/11/2024

**requirements :**
-pygame

**instructions:**
-launch main.py

**inputs :**

**-F** : Select another molecule

**-E** : move up-left

**-T** : move up-right

**-C** : move down-left

**-B** : move down-right

visualisation:
E   T
  F
C   B

**-esc** : back to main menu
